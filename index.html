quick aside

go where 呢 (ne)

呢: a Chinese modal particle

"what about...?"

gowherene is pronounced gowherene, because, like many other singlish sentences, it is a mix of English and Chinese. ne is a chinese modal particle (expresses moods)
你吃了吗? Have you eaten?
吃了, 你呢? I have, and what about you?
ne is sort of used to reflect a question back on the asker sort of like when I ask my girlfriend, "where do you want to meet today?" and she goes "not sure, gowherene?"

server-side request forgery

  • Attacker guesses the url of some internal webserver
  • Abuse the main server to read internal resources

common/possible targets

  • file:///etc/passwd
  • http://169.254.169.254/latest/meta-data/
AWS allows you to query instance metadata from within a running instance on the loopback interface. Since the server queries on the attacker's behalf, even the loopback interface is now within bounds
